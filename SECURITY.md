# 安全策略

## 报告安全问题

如果你发现 Skill 中存在提示注入、恶意脚本、供应链风险、敏感信息泄露或其他尚未修复的安全问题，请使用本仓库的 [GitHub Private Vulnerability Reporting](https://github.com/ohmuyi/muyi-skills/security/advisories/new) 私下报告。

请提供受影响的 Skill、复现输入、预期与实际行为、潜在影响，以及你已验证的缓解方式。不要在公开 Issue、Discussion 或 pull request 中披露仍可被利用的细节。

一般功能缺陷、文档问题和非敏感的提示词质量问题可以通过公开 Issue 报告。
