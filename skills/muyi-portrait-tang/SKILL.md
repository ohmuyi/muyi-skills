# 生成新中式唐风双人闺蜜照提示词

## 目标

把用户给出的少量词汇扩写为一份可直接交给图像生成模型的新中式唐风双人电影人像提示词。固定描写恰好两位二十多岁的成年中国女性，保留用户明确要求，并补全缺失的场景、人物关系、服饰妆发、道具、光线、色彩、摄影质感和画面约束。

只生成提示词。不要调用图像生成工具，不要生成视频提示词，不要处理参考图编辑。

## 生成前读取

每次生成前完整读取以下两份直接引用的文件：

- [references/tang-style-guide.md](references/tang-style-guide.md)：人物、服饰、妆发、珠宝、环境、材质、光线、色彩、摄影质感和写实约束。
- [references/composition-archetypes.md](references/composition-archetypes.md)：构图原型、组合方法和一致性检查。

不要继续追踪这两份文件之外的资料。

## 工作流

### 1. 提取输入信号

按用户原始顺序识别场景、互动、构图、服饰妆发、道具、色彩、光线、空气与情绪。不要展示分类、推导或内部取舍；不要因常规信息缺失而反问。

### 2. 解析优先级与冲突

按以下顺序确定画面：

1. 用户明确要求。
2. 匹配构图原型的默认值。
3. 全局唐风风格默认值。

同级要求冲突时采用用户最后给出的要求，并删除不兼容的早期要求。例如“俯拍、低机位仰拍”最终只采用低机位仰拍。保留用户提供的每个可视化关键词；无法直接视觉化的词转译为情绪、色彩或叙事。

用户要求与全局硬约束冲突时，只吸收其可兼容的机位、开放感、光色或情绪意图。若用户要求“胸口特写”“胸部为焦点”“领口内部”“走光”或窥视角度，不让这些词决定景别、机位或焦点；改由其余有效信号确定构图，没有其他景别信号时采用默认中景至三分之二身。把“少女感”“青春感”解释为造型和氛围，人物仍明确为成年女性。把“硬锐数码感”等冲突词只转译为有秩序的构图、明确光向或鲜明明暗，不在成品中正向使用“数码”“硬锐”“锐利”等措辞，也不退出低 MTF 雾感柔光体系。

### 3. 选择构图并补全画面

从 [references/composition-archetypes.md](references/composition-archetypes.md) 选择最匹配的原型；多个信号兼容时组合使用，没有匹配项时按其中的“自适应构图”创建新构图。只输出一个确定方案，不给候选项，不使用“或者”“可选”“方案一”等表达。

依据 [references/tang-style-guide.md](references/tang-style-guide.md) 补全所有未指定槽位。每个细节必须服务于双人关系、人物造型或空间纵深，不机械堆砌华丽元素。

### 4. 编写成品提示词

按“画面目标—场景构图—人物互动—服饰妆发—环境道具—光色摄影—画面约束”展开，使用具体、可见、可执行的自然语言。不要用摄影师姓名替代视觉描述，不主动加入 `photorealistic`、`hyperrealistic`、`ultra-detailed`、`sharp focus`、`HDR` 或其他促锐词，不写模型名、API 参数、分辨率数值或生成命令。

默认输出约 1100–1600 个中文字，目标约 1450 字，最多约 1800 字。避免关键词堆叠、抽象形容词重复和跨段复述同一约束。

## 全局硬约束

- 画面始终是恰好两位二十多岁的成年中国女性；两张脸、发髻、服色、动作和姿态有明确差异，双人脸与关键手势是第一叙事层。
- 肩颈与上胸入镜时，两套宽缓浅弧低领内襦必须稳定贴合、自然承托并覆盖胸部主体；保持非窥视视角，不把胸部、领口内部或裸色区域设为第一焦点。
- 用户指定只拍面部时，省略领口、锁骨、胸口、下身和藏露描写，不为展示默认造型而扩大景别。
- 只有下身入镜时才使用“一明一隐”的分段藏露、交叠长裙与三档纱层；腰胯保持覆盖，不使用裙底视角、走光、连续大面积裸露或现代礼服式对称高开衩。
- 两套造型各自形成完整且不同的服装、纹样、髻形和材质主题。用户指定素髻、无冠饰、无项链、无珠宝、无饰品或单一色盘时完整服从，不擅自补回被排除元素。
- 冠饰、簪钗、步摇、耳坠、项链、手与道具具有可信的连接点、受力、重力、接触和遮挡关系，不悬浮、不穿插、不融合。
- 人物、衣料与环境共享雾感自然散射光、连续空气透视和低 MTF 画意胶片成像；近镜头介质明显化开，人物前水汽极薄，双人柔和可读，远景随距离逐层褪去。
- 人体、五官、五指、衣物、道具和镜面反射结构成立。背景不增加额外人物、错误面孔、现代物件、品牌标识、水印或无关文字。
- “冷艳、清冷、疏离”等气质通过表情、视线、姿态、综合色调与构图表达，不通过锐利眼神、硬眼线、高反差轮廓或局部锐化表达。
- 不创建独立的负面提示词列表；把需要避免的问题改写为成品提示词中的肯定式画面约束。

## 输出格式

严格按以下标题和顺序输出，不添加开场白、总结、解释、参数表或代码围栏：

```text
【画面目标】
[一句话定义画面类型、双人关系、核心动作和总体氛围。]

【场景与构图】
[画幅、地点、机位、角度、景别、主体位置和连续空气透视；仅在下身入镜时交代一明一隐的纱层遮挡。]

【人物与互动】
[两位成年中国女性的外貌差异、姿态、动作、视线、情绪和手部交互；仅在下身入镜时补充腿部姿态。]

【服饰妆发】
[分别完成两套有明确差异的服装、面料、纹样、妆面、发髻和可见饰品造型，并写明关键结构与连接关系。]

【环境与道具】
[建筑、帷幔、家具和少量叙事道具，包括真实材质、位置、遮挡与反射关系。]

【光线、色彩与摄影质感】
[主光与散射路径、近中远景介质梯度、综合色调、柔和焦域、低 MTF 光学与画意胶片质感。]

【画面约束】
[用肯定、明确的自然语言集中声明人数、成年身份、人体与服装结构、反射、焦点、材质和画面纯净度。]
```

只输出一份成品提示词，不输出内部推导。除非用户明确要求解释，否则不要说明构图或风格选择理由。
